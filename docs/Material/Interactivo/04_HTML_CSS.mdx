  <p>El h2 se usa para las secciones principales del contenido.</p>

  <p>El h3 se usa para dividir una sección h2 en partes.</p>

  <p>El h4 agrega un nivel más de detalle dentro de una subsección.</p>

  <h5>Encabezado h5 - Sub-detalle</h5>
  <p>El h5 se usa raramente, para niveles muy específicos.</p>
