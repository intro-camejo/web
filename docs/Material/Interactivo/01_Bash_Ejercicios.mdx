# Bash - Ejercicios

Practicá Bash resolviendo los ejercicios. Cada uno tiene dos pestañas: una con el enunciado y el esqueleto del script, y otra con la solución. Intentá resolver el ejercicio antes de mirar la respuesta.

:::tip Cómo usar estos ejercicios
Estos ejercicios son para practicar en tu terminal. Copiá el código y ejecutalo en un archivo `.sh` (no olvides darle permisos con `chmod +x`).

```bash
# 1. Creá el archivo
nano mi_script.sh

# 2. Dale permisos de ejecución
chmod +x mi_script.sh

# 3. Ejecutalo
./mi_script.sh
```
:::

### Ejercicio 4: Par o impar

Crear un script que reciba un número como parámetro y diga si es par o impar.

**Uso esperado:** `./par_impar.sh 7` → `7 es impar`

<Tabs>
  <TabItem value="ejercicio" label="✏️ Ejercicio" default>

  ```bash
  #!/bin/bash
  # Completá el script para determinar si un número es par o impar
  # Pista: usá el operador módulo (%) dentro de $(( ))
  # Si el resto de dividir por 2 es 0, es par

  ```

  </TabItem>
  <TabItem value="solucion" label="💡 Solución">

  ```bash
  #!/bin/bash
  # Verificar si un número es par o impar
  numero=$1
  resto=$(( numero % 2 ))

  if [ $resto -eq 0 ]; then
      echo "$numero es par"
  else
      echo "$numero es impar"
  fi
  ```

### Ejercicio 6: Calculadora

Crear un script que reciba 2 números y una operación (+, -, \*, /) e imprima el resultado usando `case`.

**Uso esperado:** `./calculadora.sh 10 + 5` → `Resultado: 15`

<Tabs>
  <TabItem value="ejercicio" label="✏️ Ejercicio" default>

  ```bash
  #!/bin/bash
  # Completá la calculadora usando case
  # $1 = primer número, $2 = operación, $3 = segundo número

  num1=$1
  operacion=$2
  num2=$3

  case $operacion in
      # Completá los casos para +, -, *, /
  esac
  ```

  </TabItem>
  <TabItem value="solucion" label="💡 Solución">

  ```bash
  #!/bin/bash
  # Calculadora con case
  num1=$1
  operacion=$2
  num2=$3

  case $operacion in
      "+")
          resultado=$(( num1 + num2 ))
          echo "Resultado: $resultado"
          ;;
      "-")
          resultado=$(( num1 - num2 ))
          echo "Resultado: $resultado"
          ;;
      "*")
          resultado=$(( num1 * num2 ))
          echo "Resultado: $resultado"
          ;;
      "/")
          if [ $num2 -eq 0 ]; then
              echo "Error: no se puede dividir por cero"
          else
              resultado=$(( num1 / num2 ))
              echo "Resultado: $resultado"
          fi
          ;;
      *)
          echo "Operación no válida. Usá: +, -, *, /"
          ;;
  esac
  ```

  </TabItem>
</Tabs>

### Ejercicio 9: Factorial

Crear un script que reciba un número y calcule su factorial usando un `while`.

**Uso esperado:** `./factorial.sh 5` → `El factorial de 5 es: 120`

<Tabs>
  <TabItem value="ejercicio" label="✏️ Ejercicio" default>

  ```bash
  #!/bin/bash
  # Completá el script para calcular el factorial
  # Factorial de 5 = 5 * 4 * 3 * 2 * 1 = 120
  # Usá un while que multiplique desde el número hasta 1

  numero=$1
  factorial=1

  ```

  </TabItem>
  <TabItem value="solucion" label="💡 Solución">

  ```bash
  #!/bin/bash
  # Calcular el factorial usando while
  numero=$1
  factorial=1
  i=$numero

  while [ $i -gt 0 ]; do
      factorial=$(( factorial * i ))
      i=$(( i - 1 ))
  done

  echo "El factorial de $numero es: $factorial"
  ```

  </TabItem>

### Ejercicio 10: Números primos

Crear un script que reciba un número N e imprima todos los números primos del 1 al N.

**Uso esperado:** `./primos.sh 20` → `2 3 5 7 11 13 17 19`

<Tabs>
  <TabItem value="ejercicio" label="✏️ Ejercicio" default>

  ```bash
  #!/bin/bash
  # Completá el script para mostrar números primos del 1 a N
  # Un número primo solo es divisible por 1 y por sí mismo
  # Pista: para cada número, verificá si tiene algún divisor entre 2 y el número-1

  N=$1

  ```

  </TabItem>
  <TabItem value="solucion" label="💡 Solución">

  ```bash
  #!/bin/bash
  # Mostrar números primos del 1 a N
  N=$1

  for (( num=2; num<=N; num++ )); do
      es_primo=1

      for (( i=2; i*i<=num; i++ )); do
          if [ $(( num % i )) -eq 0 ]; then
              es_primo=0
              break
          fi
      done

      if [ $es_primo -eq 1 ]; then
          echo -n "$num "
      fi
  done
  echo  # Salto de línea final
  ```

### Ejercicio 11: Contar líneas, palabras y caracteres

Crear un script que reciba un nombre de archivo y muestre la cantidad de líneas, palabras y caracteres.

**Uso esperado:** `./contar.sh archivo.txt` → `Líneas: 10, Palabras: 50, Caracteres: 300`

<Tabs>
  <TabItem value="ejercicio" label="✏️ Ejercicio" default>

  ```bash
  #!/bin/bash
  # Completá el script para contar líneas, palabras y caracteres
  # Pista: el comando "wc" cuenta líneas (-l), palabras (-w) y caracteres (-c)
  # Podés usar $( ) para capturar la salida de un comando

  archivo=$1

  ```

  </TabItem>
  <TabItem value="solucion" label="💡 Solución">

  ```bash
  #!/bin/bash
  # Contar líneas, palabras y caracteres de un archivo
  archivo=$1

  if [ ! -f "$archivo" ]; then
      echo "Error: el archivo '$archivo' no existe"
      exit 1
  fi

  lineas=$(wc -l < "$archivo")
  palabras=$(wc -w < "$archivo")
  caracteres=$(wc -c < "$archivo")

  echo "Archivo: $archivo"
  echo "Líneas: $lineas"
  echo "Palabras: $palabras"
  echo "Caracteres: $caracteres"
  ```

### Ejercicio 14: Renombrar archivos

Crear un script que reciba un directorio y renombre todos los archivos `.txt` a `.bak`.

**Uso esperado:** `./renombrar.sh mi_carpeta` → renombra `notas.txt` a `notas.bak`, etc.

<Tabs>
  <TabItem value="ejercicio" label="✏️ Ejercicio" default>

  ```bash
  #!/bin/bash
  # Completá el script para renombrar archivos .txt a .bak
  # Pista: usá un for para recorrer los archivos .txt del directorio
  # Pista: usá mv para renombrar y ${variable%.txt}.bak para cambiar la extensión

  directorio=$1

  ```

  </TabItem>
  <TabItem value="solucion" label="💡 Solución">

  ```bash
  #!/bin/bash
  # Renombrar todos los .txt a .bak en un directorio
  directorio=$1

  if [ ! -d "$directorio" ]; then
      echo "Error: el directorio '$directorio' no existe"
      exit 1
  fi

  contador=0

  for archivo in "$directorio"/*.txt; do
      # Verificar que el glob matcheó archivos reales
      if [ -f "$archivo" ]; then
          nuevo_nombre="${archivo%.txt}.bak"
          mv "$archivo" "$nuevo_nombre"
          echo "Renombrado: $archivo -> $nuevo_nombre"
          contador=$(( contador + 1 ))
      fi
  done

  echo "Total de archivos renombrados: $contador"
  ```

  </TabItem>
</Tabs>

### Ejercicio 15: Analizador de logs

Crear un script que lea un archivo de log y cuente cuántas líneas contienen "ERROR", "WARNING" e "INFO", mostrando un resumen al final.

**Uso esperado:** `./log_analyzer.sh app.log` → muestra un resumen con la cantidad de cada tipo

<Tabs>
  <TabItem value="ejercicio" label="✏️ Ejercicio" default>

  ```bash
  #!/bin/bash
  # Completá el script para analizar un archivo de log
  # Pista: usá grep -c para contar las líneas que contienen cada palabra
  # Mostrá un resumen con la cantidad de ERROR, WARNING e INFO

  archivo=$1

  ```

  </TabItem>
  <TabItem value="solucion" label="💡 Solución">

  ```bash
  #!/bin/bash
  # Analizador de logs
  archivo=$1

  if [ ! -f "$archivo" ]; then
      echo "Error: el archivo '$archivo' no existe"
      exit 1
  fi

  total=$(wc -l < "$archivo")
  errores=$(grep -c "ERROR" "$archivo")
  warnings=$(grep -c "WARNING" "$archivo")
  infos=$(grep -c "INFO" "$archivo")

  echo "=============================="
  echo "  Resumen del log: $archivo"
  echo "=============================="
  echo "  Total de líneas:  $total"
  echo "  ERROR:            $errores"
  echo "  WARNING:          $warnings"
  echo "  INFO:             $infos"
  echo "=============================="
  ```
